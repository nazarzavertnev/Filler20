Набор утилит командной строки для взаимодействия с автоматизированным сервисом портала счетов-фактур НДС

Внимание! Инсталляция simpleapp предназначена для работы только одного предприятия с одним УНП! Для каждого УНП необходимо разворачивать отдельную инсталляцию simpleapp.

Каталоги
in - каталог исходных счетов-фактур
out - каталог подписанных и отправленных счетов-фактур
recv - каталог поступивших счетов-фактур
src - программы на Visual Basic Script
xsd - XSD схемы для проверки структуры счета-фактуры

Командные файлы
env.bat - файл настроек
signAndUploadDir.bat - подписание и отправка на портал счетов-фактур из каталога in, может вызываться с параметром -d для удаления входных файлов после обработки;
updateStatusOutDir.bat - проверка статусов счетов-фактур в папке out;
receive2Dir.bat - получение поступивших счетов-фактур в папку recv;
signAndUploadRecvDir.bat - добавление второй подписи и отправка на сервис счетов-фактур из папки recv;
updateStatusRecvDir.bat - проверка статусов счетов-фактур в папке recv.

В файле настроек env.bat можно убрать комментарии (слово rem), и указать УНП организации, чтобы каждый раз не появлялось окно выбора сертификата.
Например
  set UNP=199999999
Вместо УНП можно ввести идентификатор ключа субъекта из сертификата
  set PUBLIC_KEY_ID=
Также можно указать пароль к ключу:
  set PASSWORD=.....

Если необходимо подключаться через прокси, необходимо убрать rem и вписать параметры прокси в следующие строки:
  rem URL для подключения прокси
  rem set PROXY_URL=

  rem Порт для подключения прокси
  rem set PROXY_PORT=

  rem Пользователь для подключения прокси с авторизацией
  rem set PROXY_USER=

  rem Пароль для подключения прокси с авторизацией
  rem set PROXY_PASS=

  rem Тип прокси: 1 - HTTP прокси (по умолчанию)
  rem set PROXY_TYPE=1

Если необходимо изменить таймаут сетевого чтения, необходимо убрать rem и вписать параметр в следующую строку:
  rem Таймаут сетевого чтения (секунд)
  rem set READ_TIMEOUT=300
